# 8085 Assembly Language Repository

This repository contains 8085 Assembly Language source code written and tested using the **GUISIM8085** simulator.

The repository serves as a collection of assembly language implementations for learning, practice, and academic reference. All source files are written in standard 8085 Assembly Language and are compatible with GUISIM8085.

## Requirements

- GUISIM8085 Simulator

## Getting Started

1. Clone or download this repository.
2. Open GUISIM8085.
3. Load any `.asm` file.
4. Assemble and run the program.
